🤖 Student Chatbot (Python GUI)

📌 Project Overview

This project is a simple Student Chatbot developed using Python with a graphical user interface (GUI).
It helps students by answering basic questions using predefined responses.

---

🎯 Objectives

- Provide quick answers to common student queries
- Build a user-friendly chatbot using GUI
- Understand basic chatbot logic without NLP

---

🛠️ Technologies Used

- Python
- Tkinter (for GUI)

---

⚙️ Features

- 💬 Chat-based interface
- 🖥️ Simple GUI using Tkinter
- 📚 Predefined question-answer system
- ⚡ Fast and lightweight

---

🧠 How It Works

- User enters a question in the chatbox
- The program checks predefined questions
- If matched → shows answer
- If not → displays default response

---

🚀 Installation & Setup

1. Clone the repository
2. Run the program:
   python app.py

---

📂 Project Structure

- app.py → Main GUI application
- responses.py → Predefined questions & answers
