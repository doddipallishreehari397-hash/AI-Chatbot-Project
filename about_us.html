🤖 AI Student Assistant
Project Description
This project is an AI-powered student assistant chatbot developed using Python Flask framework. It helps students access timetable information, exam schedules, notifications, and chatbot support.
Features
✅ Student Chatbot
✅ Exam Notifications
✅ Timetable Viewer
✅ Admin Notification System
✅ Dashboard Interface
✅ Previous Search History
